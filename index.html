json 2 case class

This software is design to generate Scala case class from some json.

The basic idea to start the developement is to allow fast API integration, and forget boring time of case class mapping when using it for serialization/deserialization propose.

This soft i on the GPLv3 licence and was created by the Clever Cloud team (awesome PaaS service allowing scala running fast) for internal needs.
